# Pokemon Index

1. **Clone the repository**

   ```bash
   git clone https://github.com/R-najj/pokemon-index
   cd pokemon-index
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Set up environment variables**

   ```bash
   cp .env.example .env.local
   ```
